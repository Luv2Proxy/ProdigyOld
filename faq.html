In versions later than 1-90-0, I cannot go into shops. It says to "try again later." ▼

This happens because there is no way to access the shop API that Prodigy used in those specific versions. It relied on server-side data that no longer exists in that format.

The Fix: If you want to change your hairstyle or appearance, you will need to manually edit your save file to update those values.
